# JATC
**J**ust **A**nother (very simple) **T**etris **C**lone, made just for fun.

Requires Python 2.7 and Pygame.

Input commands:
* Left arrow: move left
* Right arrow: move right
* Up arrow: rotate
* Down arrow: quick fall
* Enter/Return: pause
* M: Next music (or mute)
* N: Previous music (or mute)
